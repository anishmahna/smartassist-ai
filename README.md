## What it does
SmartAssist AI coordinates multiple specialized agents to help users manage tasks, schedules, notes, research topics, and check weather — all through natural language.

## Agents
- Primary Agent — Coordinates all other agents
- Task Agent — Add, view, complete tasks (SQLite database)
- Schedule Agent — Manage calendar and meetings
- Notes Agent — Save and retrieve notes
- Research Agent — Search and summarize any topic
- Weather Agent — Live weather data for any city

## Tech Stack
- Google Cloud Run — Serverless deployment
- Gemini 2.5 Flash — AI reasoning engine
- SQLite — Database for tasks, schedules, notes
- MCP Tools — Weather API, Research API
- Flask — Backend API
- Python — Core language
